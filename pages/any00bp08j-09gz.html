23. Rāgapeyyāla

23. Abbreviated Texts Beginning With Greed

240–266

“Rāgassa, bhikkhave, pariññāya …pe…
“For the complete understanding of greed …

parikkhayāya …
complete ending …

pahānāya …
giving up …

khayāya …
ending …

vayāya …
vanishing …

virāgāya …
fading away …

nirodhāya …
cessation …

(…) cāgāya …
giving away …

paṭinissaggāya …pe…
letting go …

ime dasa dhammā bhāvetabbā”.
these ten things should be developed.”
