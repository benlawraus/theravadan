Aṅguttara Nikāya 1 増支部経典 一集

17. Catutthavagga 第十七　一番（第四）（是第一品）

“Etadaggaṁ, bhikkhave, mama sāvakānaṁ bhikkhūnaṁ bahussutānaṁ yadidaṁ ānando. 「私の弟子僧で一番よく説教を聞いた者はアーナンダです。

… Satimantānaṁ yadidaṁ ānando. …一番記憶力が良い者はアーナンダです。

… Gatimantānaṁ yadidaṁ ānando. …一番博識な者はアーナンダです。

… Dhitimantānaṁ yadidaṁ ānando. …一番継続している者はアーナンダです。

… Upaṭṭhākānaṁ yadidaṁ ānando. …一番の側近はアーナンダです。

… Mahāparisānaṁ yadidaṁ uruvelakassapo. …一番の大会衆をもつ者はウルウェラカッサパです。

… Kulappasādakānaṁ yadidaṁ kāḷudāyī. …一番深く家族に感銘を与える者はカールダーイーです。

… Appābādhānaṁ yadidaṁ bākulo. …一番健康なのはバックラです。

… Pubbenivāsaṁ anussarantānaṁ yadidaṁ sobhito. …一番よく前世を覚えている者はソービタです。

… Vinayadharānaṁ yadidaṁ upāli. …一番よく戒律を覚えている者はウパーリです。

… Bhikkhunovādakānaṁ yadidaṁ nandako. …一番よく尼僧に助言する者はナンダカです。

… Indriyesu guttadvārānaṁ yadidaṁ nando. …一番よく五感の扉を防護する者はナンダです。

… Bhikkhuovādakānaṁ yadidaṁ mahākappino. …一番よく僧侶達を助言する者はマハーカッピナです。

… Tejodhātukusalānaṁ yadidaṁ sāgato. …一番火に関する要素に技量のある者はサーガタです。

… Paṭibhāneyyakānaṁ yadidaṁ rādho. …一番雄弁な教えで感銘を与える者はラーダです。

… Lūkhacīvaradharānaṁ yadidaṁ mogharājā”ti. …粗末な袈裟を着る者はモガラーナです。

Vaggo catuttho.
