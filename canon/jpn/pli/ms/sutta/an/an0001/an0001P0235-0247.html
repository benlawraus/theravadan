Aṅguttara Nikāya 1 増支部経典 一集

18. Pañcamavagga 第十八　一番（第五）（是第一品）

“Etadaggaṁ, bhikkhave, mama sāvikānaṁ bhikkhunīnaṁ rattaññūnaṁ yadidaṁ mahāpajāpatigotamī. 「私の弟子尼僧で一番先輩なのはマハーパジャーパティーです。

… Mahāpaññānaṁ yadidaṁ khemā. …一番の優れた智慧を持つ者はケーマです。

… Iddhimantīnaṁ yadidaṁ uppalavaṇṇā. …一番の超能力者はウッパラワンナーです。

… Vinayadharānaṁ yadidaṁ paṭācārā. …一番よく戒律を覚えている者はパターチャーラーです。

… Dhammakathikānaṁ yadidaṁ dhammadinnā. …一番よく教えを話す者はダンマディーナーです。

… Jhāyīnaṁ yadidaṁ nandā. …一番のジャーナ瞑想者はナンダーです。

… Āraddhavīriyānaṁ yadidaṁ soṇā. …一番活力がある者はソーナーです。

… Dibbacakkhukānaṁ yadidaṁ bakulā. …一番の透視能力者はサクラーです。

… Khippābhiññānaṁ yadidaṁ bhaddā kuṇḍalakesā. …一番素早く真実を見抜き理解する者はバッダー・クンダラケーサーです。

… Pubbenivāsaṁ anussarantīnaṁ yadidaṁ bhaddā kāpilānī. …一番よく前世を覚えている者はバッダー・カーピラーニです。

… Mahābhiññāpattānaṁ yadidaṁ bhaddakaccānā. …一番鋭く真実の理解を獲得した者はバッダカッチャーナーです。

… Lūkhacīvaradharānaṁ yadidaṁ kisāgotamī. …粗末な袈裟を着る者はキサーゴータミーです。

… Saddhādhimuttānaṁ yadidaṁ siṅgālakamātā”ti. …一番固い信頼を表す者はシンガーラカマーターです。

Vaggo pañcamo.
