Saṁyutta Nikāya 18.9 Verbundene Lehrreden 18.9

1. Paṭhamavagga 1. Das erste Kapitel

Dhātusutta Elemente

Sāvatthiyaṁ viharati. In Sāvatthī.

“Taṁ kiṁ maññasi, rāhula, „Was denkst du, Rāhula?

pathavīdhātu niccā vā aniccā vā”ti? Ist das Erdelement beständig oder unbeständig?“

“Aniccā, bhante” … „Unbeständig, Herr.“ …

“āpodhātu …pe… „… das Wasserelement …

tejodhātu … das Feuerelement …

vāyodhātu … das Windelement …

ākāsadhātu … das Raumelement …

viññāṇadhātu niccā vā aniccā vā”ti? Ist das Bewusstseinselement beständig oder unbeständig?“

“Aniccā, bhante” … „Unbeständig, Herr.“ …

“evaṁ passaṁ, rāhula, sutavā ariyasāvako pathavīdhātuyāpi nibbindati …pe… āpodhātuyāpi nibbindati … tejodhātuyāpi nibbindati … vāyodhātuyāpi nibbindati … ākāsadhātuyāpi nibbindati … viññāṇadhātuyāpi nibbindati …pe… „Wenn er das sieht, wird ein gebildeter edler Schüler ernüchtert vom Erdelement, vom Wasserelement, vom Feuerelement, vom Windelement, vom Raumelement und vom Bewusstseinselement. …“

pajānātī”ti.

Navamaṁ.
