Saṁyutta Nikāya 18.4
Verbundene Lehrreden 18.4

1. Paṭhamavagga
1. Das erste Kapitel

Samphassasutta
Kontakt

Sāvatthiyaṁ viharati.
In Sāvatthī.

“Taṁ kiṁ maññasi, rāhula,
„Was denkst du, Rāhula?

cakkhusamphasso nicco vā anicco vā”ti?
Ist Augenkontakt beständig oder unbeständig?“

“Anicco, bhante” …
„Unbeständig, Herr.“ …

“sotasamphasso …pe…
„… Ohrkontakt …

ghānasamphasso …
Nasenkontakt …

jivhāsamphasso …
Zungenkontakt …

kāyasamphasso …
Körperkontakt …

manosamphasso nicco vā anicco vā”ti?
Ist Geistkontakt beständig oder unbeständig?“

“Anicco, bhante” …
„Unbeständig, Herr.“ …

“evaṁ passaṁ, rāhula, sutavā ariyasāvako cakkhusamphassasmimpi nibbindati …pe… sotasamphassasmimpi nibbindati … ghānasamphassasmimpi nibbindati … jivhāsamphassasmimpi nibbindati … kāyasamphassasmimpi nibbindati … manosamphassasmimpi nibbindati;
„Wenn er das sieht, wird ein gebildeter edler Schüler ernüchtert von Augenkontakt, Ohrkontakt, Nasenkontakt, Zungenkontakt, Körperkontakt und Geistkontakt.

nibbindaṁ virajjati …pe…
Wenn er ernüchtert ist, schwindet die Leidenschaft. …“

pajānātī”ti.

Catutthaṁ.
