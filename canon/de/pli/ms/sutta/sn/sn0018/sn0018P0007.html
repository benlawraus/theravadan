Saṁyutta Nikāya 18.7 Verbundene Lehrreden 18.7

1. Paṭhamavagga 1. Das erste Kapitel

Sañcetanāsutta Absicht

Sāvatthiyaṁ viharati. In Sāvatthī.

“Taṁ kiṁ maññasi, rāhula, „Was denkst du, Rāhula?

rūpasañcetanā niccā vā aniccā vā”ti? Ist Absicht in Bezug auf Bilder beständig oder unbeständig?“

“Aniccā, bhante” … „Unbeständig, Herr.“ …

“saddasañcetanā …pe… „… Absicht in Bezug auf Töne …

gandhasañcetanā … Absicht in Bezug auf Gerüche …

rasasañcetanā … Absicht in Bezug auf Geschmäcke …

phoṭṭhabbasañcetanā … Absicht in Bezug auf Berührungen …

dhammasañcetanā niccā vā aniccā vā”ti? Ist Absicht in Bezug auf Vorstellungen beständig oder unbeständig?“

“Aniccā, bhante” … „Unbeständig, Herr.“ …

“evaṁ passaṁ, rāhula, sutavā ariyasāvako rūpasañcetanāyapi nibbindati …pe… saddasañcetanāyapi nibbindati … gandhasañcetanāyapi nibbindati … rasasañcetanāyapi nibbindati … phoṭṭhabbasañcetanāyapi nibbindati … dhammasañcetanāyapi nibbindati …pe… „Wenn er das sieht, wird ein gebildeter edler Schüler ernüchtert von der Absicht in Bezug auf Bilder, Töne, Gerüche, Geschmäcke, Berührungen und Vorstellungen. …“

pajānātī”ti.

Sattamaṁ.
