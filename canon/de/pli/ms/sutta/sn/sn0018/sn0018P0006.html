Saṁyutta Nikāya 18.6
Verbundene Lehrreden 18.6

1. Paṭhamavagga
1. Das erste Kapitel

Saññāsutta
Wahrnehmungen

Sāvatthiyaṁ viharati.
In Sāvatthī.

“Taṁ kiṁ maññasi, rāhula,
„Was denkst du, Rāhula?

rūpasaññā niccā vā aniccā vā”ti?
Ist Wahrnehmung von Bildern beständig oder unbeständig?“

“Aniccā, bhante” …
„Unbeständig, Herr.“ …

“saddasaññā …pe…
„… Wahrnehmung von Tönen …

gandhasaññā …
Wahrnehmung von Gerüchen …

rasasaññā …
Wahrnehmung von Geschmäcken …

phoṭṭhabbasaññā …
Wahrnehmung von Berührungen …

dhammasaññā niccā vā aniccā vā”ti?
Ist Wahrnehmung von Vorstellungen beständig oder unbeständig?“

“Aniccā, bhante” …
„Unbeständig, Herr.“ …

“evaṁ passaṁ, rāhula, sutavā ariyasāvako rūpasaññāyapi nibbindati …pe… saddasaññāyapi nibbindati … gandhasaññāyapi nibbindati … rasasaññāyapi nibbindati … phoṭṭhabbasaññāyapi nibbindati … dhammasaññāyapi nibbindati …pe…
„Wenn er das sieht, wird ein gebildeter edler Schüler ernüchtert von der Wahrnehmung von Bildern, Tönen, Gerüchen, Geschmäcken, Berührungen und Vorstellungen. …“

pajānātī”ti.

Chaṭṭhaṁ.
