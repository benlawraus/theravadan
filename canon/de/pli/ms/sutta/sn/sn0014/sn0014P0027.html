Saṁyutta Nikāya 14.27
Verbundene Lehrreden 14.27

3. Kammapathavagga
3. Das Kapitel über Wege zum Begehen von Taten

Dasakammapathasutta
Zehn Wege zum Begehen von Taten

Sāvatthiyaṁ viharati.
In Sāvatthī.

“Dhātusova, bhikkhave, sattā saṁsandanti samenti.
„Mönche und Nonnen, Lebewesen kommen zusammen und vereinigen sich aufgrund eines Elements:

Pāṇātipātino pāṇātipātīhi saddhiṁ saṁsandanti samenti;
die, die lebende Geschöpfe töten mit denen, die lebende Geschöpfe töten,

adinnādāyino …pe…
die, die stehlen …

kāmesumicchācārino …
die, die sexuelle Verfehlungen begehen …

musāvādino …
die, die lügen …

pisuṇavācā …
die, die entzweiende Rede gebrauchen …

pharusavācā …
die, die harte Rede gebrauchen …

samphappalāpino samphappalāpīhi saddhiṁ saṁsandanti samenti;
die, die Unsinn reden …

abhijjhāluno abhijjhālūhi saddhiṁ saṁsandanti samenti;
die Begehrlichen …

byāpannacittā byāpannacittehi saddhiṁ saṁsandanti samenti;
die Böswilligen …

micchādiṭṭhikā micchādiṭṭhikehi saddhiṁ saṁsandanti samenti.
die mit falscher Ansicht …

Pāṇātipātā paṭiviratā pāṇātipātā paṭiviratehi saddhiṁ saṁsandanti samenti;
die, die das Töten lebender Geschöpfe unterlassen …

adinnādānā paṭiviratā …pe…
die, die das Stehlen unterlassen …

kāmesumicchācārā paṭiviratā …
die, das Begehen sexueller Verfehlungen unterlassen …

musāvādā paṭiviratā …
die, die das Lügen unterlassen …

pisuṇāya vācāya …
die, die den Gebrauch entzweiender Rede unterlassen …

pharusāya vācāya …
die, die den Gebrauch harter Rede unterlassen …

samphappalāpā paṭiviratā samphappalāpā paṭiviratehi saddhiṁ saṁsandanti samenti;
die, die das Unsinnreden unterlassen …

anabhijjhāluno anabhijjhālūhi saddhiṁ saṁsandanti samenti;
die, die nicht begehrlich sind …

abyāpannacittā abyāpannacittehi saddhiṁ saṁsandanti samenti;
die, die nicht böswillig sind …

sammādiṭṭhikā sammādiṭṭhikehi saddhiṁ saṁsandanti samentī”ti.
die mit rechter Ansicht mit denen mit rechter Ansicht.“

Pañcamaṁ.
