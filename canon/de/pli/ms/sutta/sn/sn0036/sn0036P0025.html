Saṁyutta Nikāya 36.25 Verbundene Lehrreden 36.25

3. Aṭṭhasatapariyāyavagga 3. Das Kapitel über die Darlegung der Hundertacht

Ñāṇasutta Erkennen

“‘Imā vedanā’ti me, bhikkhave, pubbe ananussutesu dhammesu cakkhuṁ udapādi, ñāṇaṁ udapādi, paññā udapādi, vijjā udapādi, āloko udapādi. „‚Das ist Gefühl.‘ Dieses Sehen, dieses Erkennen, diese Weisheit, dieses Verstehen, dieses Licht ging mir auf in Bezug auf Lehren, die ich zuvor von niemand anderem gehört hatte.

‘Ayaṁ vedanāsamudayo’ti me, bhikkhave, pubbe ananussutesu dhammesu cakkhuṁ udapādi …pe… āloko udapādi. ‚Das ist der Ursprung von Gefühl.‘ …

‘Ayaṁ vedanāsamudayagāminī paṭipadā’ti me, bhikkhave, pubbe ananussutesu dhammesu cakkhuṁ udapādi …pe… ‚Das ist die Übung, die zum Ursprung von Gefühl führt.‘ …

‘ayaṁ vedanānirodho’ti me, bhikkhave, pubbe ananussutesu dhammesu cakkhuṁ udapādi …pe… ‚Das ist das Aufhören von Gefühl.‘ …

‘ayaṁ vedanānirodhagāminī paṭipadā’ti me, bhikkhave, pubbe ananussutesu dhammesu cakkhuṁ udapādi …pe… ‚Das ist die Übung, die zum Aufhören von Gefühl führt.‘ …

‘ayaṁ vedanāya assādo’ti me, bhikkhave, pubbe ananussutesu dhammesu …pe… ‚Das ist beim Gefühl die Befriedigung.‘ …

‘ayaṁ vedanāya ādīnavo’ti me, bhikkhave, pubbe ananussutesu dhammesu …pe… ‚Das ist beim Gefühl der Nachteil.‘ …

‘idaṁ kho nissaraṇan’ti me, bhikkhave, pubbe ananussutesu dhammesu cakkhuṁ udapādi, ñāṇaṁ udapādi, paññā udapādi, vijjā udapādi, āloko udapādī”ti. ‚Das ist beim Gefühl das Entrinnen.‘ Dieses Sehen, dieses Erkennen, diese Weisheit, dieses Verstehen, dieses Licht ging mir auf in Bezug auf Lehren, die ich zuvor von niemand anderem gehört hatte.“

Pañcamaṁ.
