Aṅguttara Nikāya 4.211
Nummerierte Lehrreden 4.211

22. Parisāvagga
22. Das Kapitel über Versammlungen

Parisāsutta
Versammlung

“Cattārome, bhikkhave, parisadūsanā.
„Mönche und Nonnen, Vier verderben eine Versammlung.

Katame cattāro?
Welche vier?

Bhikkhu, bhikkhave, dussīlo pāpadhammo parisadūsano;
Ein Mönch, der unmoralisch ist, von schlechtem Charakter, verdirbt eine Versammlung.

bhikkhunī, bhikkhave, dussīlā pāpadhammā parisadūsanā;
Eine Nonne …

upāsako, bhikkhave, dussīlo pāpadhammo parisadūsano;
Ein Laienmann …

upāsikā, bhikkhave, dussīlā pāpadhammā parisadūsanā.
Eine Laienfrau, die unmoralisch ist, von schlechtem Charakter, verdirbt eine Versammlung.

Ime kho, bhikkhave, cattāro parisadūsanā.
Diese Vier verderben eine Versammlung.

Cattārome, bhikkhave, parisasobhanā.
Mönche und Nonnen, Vier schmücken eine Versammlung.

Katame cattāro?
Welche vier?

Bhikkhu, bhikkhave, sīlavā kalyāṇadhammo parisasobhano;
Ein Mönch, der sittlich ist, von gutem Charakter, schmückt eine Versammlung.

bhikkhunī, bhikkhave, sīlavatī kalyāṇadhammā parisasobhanā;
Eine Nonne …

upāsako, bhikkhave, sīlavā kalyāṇadhammo parisasobhano;
Ein Laienmann …

upāsikā, bhikkhave, sīlavatī kalyāṇadhammā parisasobhanā.
Eine Laienfrau, die sittlich ist, von gutem Charakter, schmückt eine Versammlung.

Ime kho, bhikkhave, cattāro parisasobhanā”ti.
Diese Vier schmücken eine Versammlung.“

Paṭhamaṁ.
