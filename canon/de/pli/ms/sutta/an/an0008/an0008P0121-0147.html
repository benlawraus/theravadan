Aṅguttara Nikāya 8
Nummerierte Lehrreden 8.121–147

11. Rāgapeyyāla
11. Abgekürzte Texte, die mit Gier beginnen

~
Lehrreden ohne Titel über Gier

“Rāgassa, bhikkhave, pariññāya …pe…
„Um Gier vollständig zu verstehen …

parikkhayāya …
sie vollständig aufzulösen …

pahānāya …
sie aufzugeben …

khayāya …
sie aufzulösen …

vayāya …
sie zum Verschwinden zu bringen …

virāgāya …
zum Schwinden …

nirodhāya …
zum Aufhören …

cāgāya …
um sie herzugeben …

paṭinissaggāya …pe…
um sie loszulassen …

ime aṭṭha dhammā bhāvetabbā”.
müssen diese acht Dinge entwickelt werden.“
