Aṅguttara Nikāya 10
Nummerierte Lehrreden 10.240–266

23. Rāgapeyyāla
23. Abgekürzte Texte, die mit Gier beginnen

~
Lehrreden ohne Titel über Gier

“Rāgassa, bhikkhave, pariññāya …pe…
„Um Gier vollständig zu verstehen …

parikkhayāya …
sie vollständig aufzulösen …

pahānāya …
sie aufzugeben …

khayāya …
sie aufzulösen …

vayāya …
sie zum Verschwinden zu bringen …

virāgāya …
zum Schwinden …

nirodhāya …
zum Aufhören …

(…) cāgāya …
um sie herzugeben …

paṭinissaggāya …pe…
um sie loszulassen …

ime dasa dhammā bhāvetabbā”.
müssen diese zehn Dinge entwickelt werden.“
