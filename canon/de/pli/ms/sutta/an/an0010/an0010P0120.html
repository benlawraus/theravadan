Aṅguttara Nikāya 10.120
Nummerierte Lehrreden 10.120

12. Paccorohaṇivagga
12. Das Kapitel über die Abstiegszeremonie

Dutiyapaccorohaṇīsutta
Die Abstiegszeremonie (2)

“Ariyaṁ vo, bhikkhave, paccorohaṇiṁ desessāmi.
„Mönche und Nonnen, ich will euch den edlen Abstieg lehren.

Taṁ suṇātha …
Hört zu und gebraucht den Geist gut, ich werde sprechen. …

katamā ca, bhikkhave, ariyā paccorohaṇī?
Und was ist der edle Abstieg?

Idha, bhikkhave, ariyasāvako iti paṭisañcikkhati:
Da überlegt ein edler Schüler:

‘micchādiṭṭhiyā kho pāpako vipāko—
‚Falsche Ansicht führt zu einem schlechten Ergebnis

diṭṭhe ceva dhamme abhisamparāyañcā’ti.
sowohl in diesem als im nächsten Leben.‘

So iti paṭisaṅkhāya micchādiṭṭhiṁ pajahati;
Wenn er so nachdenkt, gibt er falsche Ansicht auf,

micchādiṭṭhiyā paccorohati.
er steigt von falscher Ansicht herab.

Micchāsaṅkappassa kho pāpako vipāko …
‚Falsches Denken führt zu einem schlechten Ergebnis …‘ …

micchāvācāya kho …
‚Falsche Rede …‘ …

micchākammantassa kho …
‚Falsches Verhalten …‘ …

micchāājīvassa kho …
‚Falscher Lebenserwerb …‘ …

micchāvāyāmassa kho …
‚Falscher Einsatz …‘ …

micchāsatiyā kho …
‚Falsche Achtsamkeit …‘ …

micchāsamādhissa kho …
‚Falsche Versenkung …‘ …

micchāñāṇassa kho …
‚Falsche Erkenntnis …‘ …

micchāvimuttiyā kho pāpako vipāko—
‚Falsche Freiheit führt zu einem schlechten Ergebnis

diṭṭhe ceva dhamme abhisamparāyañcāti.
sowohl in diesem als im nächsten Leben.‘

So iti paṭisaṅkhāya micchāvimuttiṁ pajahati;
Wenn er so nachdenkt, gibt er falsche Freiheit auf,

micchāvimuttiyā paccorohati.
er steigt von falscher Freiheit herab.

Ayaṁ vuccati, bhikkhave, ariyā paccorohaṇī”ti.
Das nennt man den edlen Abstieg.“

Aṭṭhamaṁ.
