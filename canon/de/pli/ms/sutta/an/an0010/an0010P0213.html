Aṅguttara Nikāya 10.213 Nummerierte Lehrreden 10.213

21. Karajakāyavagga 21. Das Kapitel über den aus Taten geborenen Körper

Mātugāmasutta Eine Frauensperson

“Dasahi, bhikkhave, dhammehi samannāgato mātugāmo yathābhataṁ nikkhitto evaṁ niraye. „Eine Frauensperson, die zehn Eigenschaften besitzt, wird in die Hölle gestoßen.

Katamehi dasahi? Welche zehn?

Pāṇātipātī hoti …pe… Sie tötet lebende Geschöpfe …

adinnādāyī hoti … Sie stiehlt …

kāmesumicchācārī hoti … Sie begeht sexuelle Verfehlungen …

musāvādī hoti … Sie lügt …

pisuṇavāco hoti … Sie redet so, dass sie andere entzweit …

pharusavāco hoti … Sie redet hart …

samphappalāpī hoti … Sie redet Unsinn …

abhijjhālu hoti … Sie ist begehrlich …

byāpannacitto hoti … Sie hegt bösen Willen …

micchādiṭṭhiko hoti …. Sie hat falsche Ansicht …

Imehi kho, bhikkhave, dasahi dhammehi samannāgato mātugāmo yathābhataṁ nikkhitto evaṁ niraye. Eine Frauensperson, die diese zehn Eigenschaften besitzt, wird in die Hölle gestoßen.

Dasahi, bhikkhave, dhammehi samannāgato mātugāmo yathābhataṁ nikkhitto evaṁ sagge. Eine Frauensperson, die zehn Eigenschaften besitzt, wird in den Himmel erhoben.

Katamehi dasahi? Welche zehn?

Pāṇātipātā paṭivirato hoti …pe… Sie unterlässt das Töten lebender Geschöpfe …

adinnādānā paṭivirato hoti … Sie unterlässt das Stehlen …

kāmesumicchācārā paṭivirato hoti … Sie unterlässt sexuelle Verfehlungen …

musāvādā paṭivirato hoti … Sie unterlässt das Lügen …

pisuṇāya vācāya paṭivirato hoti … Sie unterlässt entzweiende Rede …

pharusāya vācāya paṭivirato hoti … Sie unterlässt harte Rede …

samphappalāpā paṭivirato hoti … Sie unterlässt das Unsinnreden …

anabhijjhālu hoti … Sie ist zufrieden …

abyāpannacitto hoti … Sie hat ein gütiges Herz …

sammādiṭṭhiko hoti …. Sie hat rechte Ansicht …

Imehi kho, bhikkhave, dasahi dhammehi samannāgato mātugāmo yathābhataṁ nikkhitto evaṁ sagge”ti. Eine Frauensperson, die diese zehn Eigenschaften besitzt, wird in den Himmel erhoben.“

Tatiyaṁ.
