Aṅguttara Nikāya 10.168 Nummerierte Lehrreden 10.168

17. Jāṇussoṇivagga 17. Das Kapitel mit Jānussoṇi

Ariyapaccorohaṇīsutta Der edle Abstieg

“Ariyaṁ vo, bhikkhave, paccorohaṇiṁ desessāmi. „Mönche und Nonnen, ich will euch den edlen Abstieg lehren.

Taṁ suṇātha, sādhukaṁ manasi karotha, bhāsissāmī”ti. Hört zu und gebraucht den Geist gut, ich werde sprechen.“

“Evaṁ, bhante”ti kho te bhikkhū bhagavato paccassosuṁ. „Ja, Herr“, antworteten sie.

Bhagavā etadavoca: Der Buddha sagte:

“Katamā ca, bhikkhave, ariyā paccorohaṇī? „Und was, Mönche und Nonnen, ist der edle Abstieg?

Idha, bhikkhave, ariyasāvako iti paṭisañcikkhati: Da überlegt ein edler Schüler:

‘pāṇātipātassa kho pāpako vipāko—‚Das Töten lebender Geschöpfe führt zu einem schlechten Ergebnis

diṭṭhe ceva dhamme abhisamparāyañcā’ti. sowohl in diesem als im nächsten Leben.‘

So iti paṭisaṅkhāya pāṇātipātaṁ pajahati; Wenn er so nachdenkt, gibt er das Töten lebender Geschöpfe auf,

pāṇātipātā paccorohati. er steigt vom Töten lebender Geschöpfe herab.

… ‘Adinnādānassa kho pāpako vipāko—… ‚Stehlen führt zu einem schlechten Ergebnis

diṭṭhe ceva dhamme abhisamparāyañcā’ti. sowohl in diesem als im nächsten Leben.‘

So iti paṭisaṅkhāya adinnādānaṁ pajahati; Wenn er so nachdenkt, gibt er das Stehlen auf,

adinnādānā paccorohati. er steigt vom Stehlen herab.

… ‘Kāmesumicchācārassa kho pāpako vipāko …pe… … ‚Sexuelle Verfehlungen führen zu einem schlechten Ergebnis …‘ …

kāmesumicchācārā paccorohati. er steigt von sexuellen Verfehlungen herab.

… ‘Musāvādassa kho pāpako vipāko …pe… … ‚Lügen führt zu einem schlechten Ergebnis …‘ …

musāvādā paccorohati. er steigt vom Lügen herab.

… ‘Pisuṇāya vācāya kho pāpako vipāko …pe… … ‚Entzweiende Rede führt zu einem schlechten Ergebnis …‘ …

pisuṇāya vācāya paccorohati. er steigt von entzweiender Rede herab.

… ‘Pharusāya vācāya kho pāpako vipāko …pe… … ‚Harte Rede führt zu einem schlechten Ergebnis …‘ …

pharusāya vācāya paccorohati. er steigt von harter Rede herab.

… ‘Samphappalāpassa kho pāpako vipāko …pe… … ‚Unsinnreden führt zu einem schlechten Ergebnis …‘ …

samphappalāpā paccorohati. er steigt vom Unsinnreden herab.

… ‘Abhijjhāya kho pāpako vipāko …pe… … ‚Begehrlichkeit führt zu einem schlechten Ergebnis …‘ …

abhijjhāya paccorohati. er steigt von der Begehrlichkeit herab.

… ‘Byāpādassa kho pāpako vipāko …pe… … ‚Böser Wille führt zu einem schlechten Ergebnis …‘ …

byāpādā paccorohati. er steigt von bösem Willen herab.

… ‘Micchādiṭṭhiyā kho pāpako vipāko diṭṭhe ceva dhamme abhisamparāyañcā’ti. … ‚Falsche Ansicht führt zu einem schlechten Ergebnis sowohl in diesem als im nächsten Leben.‘

So iti paṭisaṅkhāya micchādiṭṭhiṁ pajahati; Wenn er so nachdenkt, gibt er falsche Ansicht auf,

micchādiṭṭhiyā paccorohati. er steigt von falscher Ansicht herab.

Ayaṁ vuccati, bhikkhave, ariyā paccorohaṇī”ti. Das nennt man den edlen Abstieg.“

Dutiyaṁ.
