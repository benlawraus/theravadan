Aṅguttara Nikāya 1 Nummerierte Lehrreden 1.378–393

28. Pasādakaradhammavagga 28. Das Kapitel über ergreifende Eigenschaften

“Addhamidaṁ, bhikkhave, lābhānaṁ yadidaṁ āraññikattaṁ …pe… „Mönche und Nonnen, das ist zweifellos etwas, das sich lohnt: nämlich in der Wildnis leben …

piṇḍapātikattaṁ … nur Almosen essen …

paṁsukūlikattaṁ … Fetzenroben tragen …

tecīvarikattaṁ … nur drei Roben besitzen …

dhammakathikattaṁ … den Dhamma lehren …

vinayadharattaṁ … das Ordensrecht im Gedächtnis haben …

bāhusaccaṁ … sehr gelehrt sein …

thāvareyyaṁ … angesehen sein …

ākappasampadā … eine ordentliche Erscheinung haben …

parivārasampadā … eine Gefolgschaft haben …

mahāparivāratā … eine große Gefolgschaft haben …

kolaputti … aus einer guten Familie stammen …

vaṇṇapokkharatā … gut aussehen …

kalyāṇavākkaraṇatā … sich gut ausdrücken …

appicchatā … Genügsamkeit …

appābādhatā”ti. bei guter Gesundheit sein.“

(Soḷasa pasādakaradhammā niṭṭhitā.)
