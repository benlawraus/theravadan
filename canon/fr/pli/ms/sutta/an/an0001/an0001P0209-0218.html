Aṅguttara Nikāya 1
Discours Sous Forme De Listes 1.209–218

16. Tatiyavagga
16. Troisième groupe

“Etadaggaṁ, bhikkhave, mama sāvakānaṁ bhikkhūnaṁ sikkhākāmānaṁ yadidaṁ rāhulo.
209. Mendiants, le plus distingué parmi mes disciples mendiants en ce qui concerne le désir de l’entraînement est Rahoula.

… Saddhāpabbajitānaṁ yadidaṁ raṭṭhapālo.
210. Mendiants, le plus distingué parmi mes disciples mendiants ayant quitté la vie de foyer par conviction est Ratthapala.

… Paṭhamaṁ salākaṁ gaṇhantānaṁ yadidaṁ kuṇḍadhāno.
211. Mendiants, le plus distingué parmi mes disciples mendiants qui sont les premiers à prendre un ticket-repas est Koundadhano.

… Paṭibhānavantānaṁ yadidaṁ vaṅgīso.
212. Mendiants, le plus distingué parmi mes disciples mendiants en ce qui concerne le sens de la répartie est Vanguissa.

… Samantapāsādikānaṁ yadidaṁ upaseno vaṅgantaputto.
213. Mendiants, le plus distingué parmi mes disciples mendiants qui inspirent la confiance à tous est Oupaséna Vangantapoutta.

… Senāsanapaññāpakānaṁ yadidaṁ dabbo mallaputto.
214. Mendiants, le plus distingué parmi mes disciples mendiants qui attribuent les abris est Dabba fils de Malla.

… devatānaṁ piyamanāpānaṁ yadidaṁ pilindavaccho.
215. Mendiants, le plus distingué parmi mes disciples mendiants agréables & plaisants aux dévas est Pilindavaccha.

… Khippābhiññānaṁ yadidaṁ bāhiyo dārucīriyo.
216. Mendiants, le plus distingué parmi mes disciples mendiants en ce qui concerne la connaissance directe rapide est Bahiya Daroutchiriya.

… Cittakathikānaṁ yadidaṁ kumārakassapo.
217. Mendiants, le plus distingué parmi mes disciples mendiants qui ont des mots d’esprit est Koumarakassapa.

… Paṭisambhidāpattānaṁ yadidaṁ mahākoṭṭhito”ti.
218. Mendiants, le plus distingué parmi mes disciples mendiants ayant atteint les connaissances analytiques est Mahakotthita.

Vaggo tatiyo.
