Aṅguttara Nikāya 1 Discours Sous Forme De Listes 1.378–393

28. Pasādakaradhammavagga 28. Les choses qui inspirent la confiance

“Addhamidaṁ, bhikkhave, lābhānaṁ yadidaṁ āraññikattaṁ …pe… 378. Véritablement, mendiants, ceci est une acquisition : être quelqu’un qui vit en forêt.

piṇḍapātikattaṁ … 379. Véritablement, mendiants, ceci est une acquisition : être quelqu’un qui se nourrit de nourriture d’aumônes.

paṁsukūlikattaṁ … 380. Véritablement, mendiants, ceci est une acquisition : être quelqu’un qui porte des haillons abandonnés comme déchets.

tecīvarikattaṁ … 381. Véritablement, mendiants, ceci est une acquisition : être quelqu’un qui porte seulement trois robes.

dhammakathikattaṁ … 382. Véritablement, mendiants, ceci est une acquisition : être un orateur sur le Dhamma.

vinayadharattaṁ … 383. Véritablement, mendiants, ceci est une acquisition : être quelqu’un qui connaît la Discipline par cœur.

bāhusaccaṁ … 384. Véritablement, mendiants, ceci est une acquisition : être quelqu’un de très instruit.

thāvareyyaṁ … 385. Véritablement, mendiants, ceci est une acquisition : être un ancien.

ākappasampadā … 386. Véritablement, mendiants, ceci est une acquisition : être bien présenté.

parivārasampadā …

mahāparivāratā …

kolaputti … 389. Véritablement, mendiants, ceci est une acquisition : être d’une bonne famille.

vaṇṇapokkharatā … 390. Véritablement, mendiants, ceci est une acquisition : avoir beau teint.

kalyāṇavākkaraṇatā … 391. Véritablement, mendiants, ceci est une acquisition : être un bon orateur.

appicchatā … 392. Véritablement, mendiants, ceci est une acquisition : avoir peu de désirs.

appābādhatā”ti. 393. Véritablement, mendiants, ceci est une acquisition : être en bonne santé.

(Soḷasa pasādakaradhammā niṭṭhitā.)
